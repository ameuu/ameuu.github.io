FastJson入门

Fastjson是Alibaba维护的开源JSON解析库，其优势是”快”。它可以解析 JSON 格式的字符串，⽀持将 Java Bean 序列 化为 JSON 字符串，也可以从JSON字符串反序列化到 Java Bean 。

实现调用setter

小结：

在分析JSON.parse的时候可以知道：

调用setter满足的条件：

  • 方法名长度大于4并且以set开始
  • 不是静态方法
  • 返回值为空或者当前类
  • 参数个数为1

调用getter满足的条件：

  • 方法名长度大于4且以get开始
  • 不是静态方法
  • 返回值为集合等
  • 无参数
  • 第四个字符是大写的
